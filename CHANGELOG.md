# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-02-14

### Initial Release

#### Features
- 🎯 Tap or long-press to inspect any widget
- 📊 Enhanced property extraction (constraints, padding, text styles, borders, shadows)
- 🌳 Widget tree navigation (parent, children, ancestors)
- 📋 Copy to clipboard (single property or all)
- 📤 Export inspection data as JSON
- 🔙 History navigation (back/forward through inspections)
- 🔍 Search and filter properties
- 📑 Organized tabs (Properties, Layout, Styles, Tree)
- 🎨 Visual overlay with green highlight
- 🔄 Toggle button to enable/disable inspection
- 🎯 Debug-only mode (automatically disabled in release builds)
- 🚀 Zero external dependencies
- ⚡ Minimal performance impact (~50KB in debug, 0KB in release)

#### Developer Experience
- Simple integration - wrap MaterialApp with FlutterLens
- Works globally across all app screens
- Configurable trigger modes (tap vs long-press)
- Professional UI with Material Design
- Comprehensive documentation and examples
